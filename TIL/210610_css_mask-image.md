```
  mask-image: linear-gradient(
    90deg,
    transparent calc(50% - 33.5rem)
    black calc(50% - 27.5rem),
    black calc(50% + 27.5rem),
    transparent calc(50% + 33.5rem)
  );
```

알파채널을 조절하는 속성.
transparent를 준 영역은 투명해지고, black이나 white를 준 영역은 원본 이미지가 나온다.
mask-mode라는 속성에 영향을 받는다고 하는데 이건 따로 알아볼 예정.
